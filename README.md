# Brain-Tumors-CNN
CNN for identifying brain scans with brain tumors

btcnn.py is the file used to train, export, and evaluate a model.
play.py allows the user to use the model to test on individual images in the testing dataset. This script was made on request from a friend because they wanted to experiment with the model.

Results:
accuracy: 0.9542 - loss: 0.2088
